pfts.util.general.bump_version(part: str, testing_mode: bool = False) → str¶

Increments either the major, minor, or patch version of the pfts module.

Parameters:

  • part (str) – Which part of the version to increment.

  • testing_mode (bool, optional) – Whether to run in debug mode or not, defaults to False

Returns:

If debugging, return new version info. Otherwise return nothing.

str|None
